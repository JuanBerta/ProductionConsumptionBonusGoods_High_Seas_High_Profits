# Show Production & Consumption of Bonus Goods on World Map Cities

A UI enhancement mod for **High Seas, High Profits!** that displays the real-time **production** and **consumption** values of each bonus good directly on city labels in the world map.

## 🚀 Features

### Real-Time Production & Consumption Data
See how much of each bonus good a city produces and consumes without opening extra menus.

This helps you:

- Quickly compare cities at a glance  
- Spot shortages before they affect prices  
- Find profitable trade opportunities  
- Avoid surprises when consumption is higher than production  

## 🛠 Installation

### Requirements
- **MelonLoader** installed

### Steps

1. Download the latest `ShowProductionConsumption.dll` from the **Releases** page.  
2. Open your game installation folder:  

   `SteamLibrary\steamapps\common\High Seas, High Profits!`

3. Place `ShowProductionConsumption.dll` inside the `Mods` folder.  
4. Launch the game.

## 📌 Notes

- Make sure you are using a compatible version of **MelonLoader**.
- If the `Mods` folder does not exist, create it manually.
